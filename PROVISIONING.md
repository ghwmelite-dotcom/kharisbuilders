# Provisioning checklist — Kharis Builders

Generated by `scripts/new-church.mjs`. Run these from the repo root, in order.
Each `wrangler` call may prompt you to log in first (`npx wrangler login`).

> **Fast path:** run `node scripts/new-church.mjs --provision` (with `CLOUDFLARE_ACCOUNT_ID` set)
> to do steps 1–5 automatically. Preview first with `--provision --dry-run`.

## 1. Create Cloudflare resources

```bash
npx wrangler d1 create kharisbuilders
#   -> paste the printed database_id into wrangler.jsonc (replace PASTE_FROM_D1_CREATE)
npx wrangler kv namespace create SESSION
#   -> paste the printed id into wrangler.jsonc (replace PASTE_FROM_KV_CREATE)
npx wrangler r2 bucket create kharisbuilders-media
npx wrangler vectorize create kharisbuilders-sermons --dimensions=768 --metric=cosine
```

## 2. Migrate + seed the database

```bash
npx wrangler d1 migrations apply kharisbuilders --remote
npx wrangler d1 execute kharisbuilders --remote --file db/seed.sql
npx wrangler d1 execute kharisbuilders --remote --file db/seed_funds.sql
npx wrangler d1 execute kharisbuilders --remote --file db/seed_lists.sql
```

## 3. Secrets

```bash
npx wrangler secret put TURNSTILE_SECRET_KEY   # spam protection on visit/registration/giving forms
npx wrangler secret put PAYSTACK_SECRET_KEY    # online giving
npx wrangler secret put RESEND_API_KEY         # optional: staff email notifications
```

## 4. Cloudflare Access (protect the admin) — manual, in the dashboard

Zero Trust → Access → Applications → Add a **self-hosted** app of type **Workers**:
- Cover BOTH paths on your worker hostname: `admin` AND `api/admin`.
- Policy: Allow → your staff emails (One-time PIN or Google).

## 5. Deploy

```bash
npm run deploy
```

## 6. Post-deploy (Admin → Settings)

- Giving: set `paystack_public_key`, `currency=GHS`, `giving_enabled=true`; set the Paystack webhook to `https://kharisbuilders.workers.dev/api/webhooks/paystack`.
- Turnstile: create a widget for your hostname → put the secret (step 3) + set `turnstile_site_key`.
- AI search: Admin → Sermons → "Reindex AI search" once sermons exist.
- Replace placeholder images and copy throughout the admin.
